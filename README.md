## ✨ Overview

A modern, high-performance developer portfolio built for **Shiva Sai Poloju**. Designed with glassmorphism aesthetics, dynamic dark mode styling, mouse-reactive particle physics, 3D Coverflow project rail, and automated email contact integrations.

---

## 🌟 Key Features

- **🎛️ 3D Coverflow Project Rail**: Interactive 3D perspective slider supporting drag/swipe gestures, depth blur filters, keyboard arrow navigation, and real-time project snapshot overlays.
- **🚑 Flagship Featured Project**: Showcases **LifeLine AI** (Disaster-Resilient Offline Medical Assistant) featuring custom architecture flowcharts and system diagrams.
- **🛠️ 35+ Categorized Technical Skills**: Filterable skill tree organized across 7 categories (*Languages, Frontend, Backend, Databases, AI & ML, Tools & Platforms, Core Concepts*) with custom vector SVG logos and brand glow effects.
- **🌌 Interactive Particle Network**: Custom HTML5 Canvas physics background responding dynamically to mouse proximity and particle repulsion forces.
- **📬 Direct Inbox Contact Form**: Instant background HTTP submission sending messages straight to `shivasaipoloju5@gmail.com` with celebratory confetti effects and mailto fallback.

---

## 🛠️ Technology Stack

- **Core Framework**: React 18 + Vite
- **Styling**: Tailwind CSS + Custom CSS Glassmorphic Tokens
- **Animations**: Framer Motion (3D Transforms, AnimatePresence, Smooth Scroll)
- **Effects**: Canvas Confetti & Custom HTML5 Physics Canvas
- **Icons**: Custom Vector SVGs & Lucide React
- **Deployment**: Vercel

## 💻 Local Development & Installation

### Prerequisites
- **Node.js**: `v18.x` or higher
- **npm** or **yarn**

### Steps

1. **Clone the repository**:
   ```bash
   git clone https://github.com/shivapoloju/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start local development server**:
   ```bash
   npm run dev
   ```
   *Open `http://localhost:5173/` in your browser.*

4. **Build for production**:
   ```bash
   npm run build
   ```
